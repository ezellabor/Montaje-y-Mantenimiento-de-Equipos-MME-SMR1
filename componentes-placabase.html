Componentes principales de la placa base

Tabla profesional con función, especificaciones actuales, parámetros clave y evolución tecnológica

Componente Función Especificaciones actuales Parámetros clave Evolución
Microprocesador (CPU) Procesar instrucciones y datos del sistema x86-64, 3–5 GHz, hasta 16 núcleos Cores, hilos, caché, TDP, turbo Pentium → Core → Ryzen → Ryzen 9
Memoria RAM Almacenar temporalmente datos e instrucciones DDR5, 5200–8000 MHz, 8–64 GB Capacidad, frecuencia, latencia, canales DDR3 → DDR4 → DDR5
Chipset Coordinar comunicación entre CPU y periféricos PCIe 5.0, USB-C, NVMe, OC Líneas PCIe, puertos USB/SATA, soporte OC Integrado → PCIe 4.0 → PCIe 5.0
Memoria secundaria Almacenamiento permanente de la información NVMe SSD 1–4 TB, 3500–7000 MB/s Capacidad, interfaz, lectura/escritura HDD → SATA SSD → NVMe SSD
Tarjetas de expansión Ampliar capacidades: GPU, red, sonido, captura PCIe 4.0/5.0, NIC 10 GbE, GPU dedicada Ancho PCIe, consumo, compatibilidad PCI → PCIe 3.0 → PCIe 5.0
BIOS / UEFI Inicializar hardware y cargar configuraciones UEFI GUI, TPM 2.0, Secure Boot Modo Legacy/UEFI, actualizaciones BIOS → UEFI
CMOS Guardar ajustes del sistema de arranque CMOS + batería CR2032 Fecha/hora, arranque, parámetros base CMOS tradicional → NVRAM
PSU (Fuente de alimentación) Suministrar energía estable al sistema ATX 650–1200 W, 80 PLUS Gold/Platinum Potencia, eficiencia, conectores No certificada → Bronze → Platinum/Titanium
Las especificaciones son orientativas y dependen del fabricante y modelo.
