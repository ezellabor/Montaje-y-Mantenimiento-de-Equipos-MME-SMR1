Componentes Principales de una Placa Base

Nombre componente Función Especificaciones actuales Parámetros clave Evolución
Microprocesador (CPU) Ejecuta las instrucciones del sistema y de las aplicaciones Intel 12-14 Gen / AMD Ryzen 5–9 núcleos, hilos, frecuencia, caché Core i7 → Core i9 / Ryzen 5 → Ryzen 9
RAM Almacena datos temporales para acceso rápido DDR5 (4800–6400 MHz) frecuencia, latencia, capacidad, canales DDR3 → DDR4 → DDR5
Chipset Controla la comunicación entre los componentes Z790 / B650 / X670 conectividad, PCIe, VRM, puertos Z690 → Z790 / X570 → X670
Memoria secundaria Almacena datos permanentes SSD NVMe PCIe 4.0 / 5.0 velocidad lectura/escritura, TBW, interfaz HDD → SATA SSD → NVMe PCIe 4.0 → 5.0
Tarjetas de expansión Añaden funciones adicionales al sistema PCIe 4.0 / 5.0 — GPU RTX, NIC 10GbE, WiFi 6E n.º de líneas PCIe, ancho de banda, consumo PCI → PCIe 3.0 → PCIe 4.0 → 5.0
BIOS / UEFI Inicializa el hardware y carga el sistema operativo UEFI con Secure Boot modo de arranque, soporte NVMe, TPM BIOS → UEFI
CMOS Almacena configuración básica del sistema Memoria CMOS + Batería CR2032 hora/fecha, arranque, voltajes Mantenimiento de CMOS clásico → Config CMOS en UEFI
PSU (Fuente alimentación) Suministra energía estable al sistema ATX 750–1200W 80+ Gold / Platinum vatios, eficiencia, protecciones, modular Bronze → Gold → Platinum → Titanium
