# Compatible AMD SVM

**Compatible AMD SVM** significa que un procesador o equipo es compatible con la tecnología **AMD Secure Virtual Machine (SVM)**.

## ¿Qué es SVM?

SVM es una **tecnología de virtualización por hardware** desarrollada por AMD que permite ejecutar máquinas virtuales de forma más eficiente.

Es el equivalente a **Intel VT-x** en procesadores Intel.

## ¿Para qué sirve?

Permite:

- Ejecutar varios sistemas operativos al mismo tiempo (Windows, Linux, etc.).
- Utilizar software de virtualización como:
  - VirtualBox
  - VMware
  - Hyper-V
- Usar emuladores (por ejemplo, Android Studio).
- Crear entornos de prueba o laboratorio (muy usado en ciberseguridad y formación).

## ⚠️ Importante

Aunque el equipo sea compatible, normalmente es necesario:

- Acceder a la BIOS/UEFI.
- Activar la opción **SVM Mode** (o similar).

## Resumen

> **Compatible AMD SVM = tu procesador soporta virtualización avanzada, pero puede requerir activación en BIOS/UEFI.**
