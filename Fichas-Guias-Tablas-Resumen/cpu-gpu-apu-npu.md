# CPU vs GPU vs APU vs NPU

| Componente | Nombre Completo | Función Principal |
|------------|-----------------|-------------------|
| **CPU** | _Central Processing Unit_ | Procesamiento general |
| **GPU** | _Graphics Processing Unit_ | Procesamiento gráfico/paralelo |
| **APU** | _Accelerated Processing Unit_ | CPU + GPU en un solo chip |
| **NPU** | _Neural Processing Unit_ | Procesamiento neuronal |  

# Comparativa

| Componente | Arquitectura | Especialización | Ventajas | Desventajas | Ejemplos |
|------------|--------------|-----------------|----------|-------------|----------|
| **CPU** | - Pocos núcleos complejos<br>- Alta frecuencia<br>- Cache grande | Procesamiento secuencial | - Máxima versatilidad<br>- Buen single-thread<br>- Baja latencia | - Limitado en paralelismo<br>- Mayor consumo por core | Intel Core i7, AMD Ryzen 9 |
| **GPU** | - Miles de núcleos simples<br>- Arquitectura paralela<br>- Memoria dedicada | Cálculos masivamente paralelos | - Alto throughput<br>- Ideal para gráficos<br>- Eficiente en ML | - Pobre en tareas secuenciales<br>- Alto consumo | NVIDIA RTX 4090, AMD RX 7900 |
| **APU** | - CPU + GPU en un chip<br>- Memoria compartida<br>- Diseño integrado | Solución todo-en-uno | - Eficiencia energética<br>- Menor costo<br>- Compacto | - Rendimiento limitado<br>- Memoria compartida | AMD Ryzen 7 5700G, Intel Iris Xe |
| **NPU** | - Circuitos especializados<br>- Baja precisión<br>- Baja latencia | Algoritmos de IA/ML | - Bajo consumo<br>- Alta eficiencia en IA<br>- Procesamiento en tiempo real | - Uso muy específico<br>- Poca versatilidad | Apple Neural Engine, Intel NPU |
